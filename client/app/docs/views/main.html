  • Buttons

    For basic button, just add class="btn" to the target element.

    There are also few other button colors reached by classes btn-light , btn-info , btn-success , btn-danger , btn-inverse
    Button sizes

    Different button sizes are reached by classes btn-mini , btn-small , btn-medium , btn-large

    For full width (block) button use class btn-full
    Button states

    For disabled state add a class disabled

    For toggle button use class active
    Nav buttons

    Nav buttons are commonly used for second navigation level (page control). They are placed within button-stack.

    For blue-colored button add class navbtn-info.

    Nav buttons have auto width, but for strict width (currently 48px) use class strict. This is useful when icon is used as button content.

    There is specific nav button meant to be used for adding/creating action. To get this button use class sd-create-button.

    Modal state (open/close) is handled by data-model="" attribute. 

                                scope.modalActive = false;

                            scope.openModal = function() {
                                scope.modalActive = true;
                            };

                            scope.closeModal = function() {
                                scope.modalActive = false;
                            };

    Modal content itself is transcluded. Parts of modal are modal-header, modal-body and modal-footer. If presented, those parts have to be placed in the first level in html. Otherwise custom styling is needed.

    Modal sizes

    By default modal has static width (currently 460px) and auto height. If the content is bigger than the screen height, content (modal-body) will become scrollable.

    There are several size variations of modal window:

    • Basic modal window.

    • Basic modal with lot of content. Modal body is scrollable.

      This happens by default.

    • Streched modal is always streched (vertically) to the screen limits.

      Use class modal-strech to achieve this.

    • Static modal has static body height no matter how large the content is. Bigger content will make box scrollable.

      Use class modal-static to achieve this.

    • Large modal has larger width than default one. (currently 660px)

      Use class modal-large to achieve this.

    • Big modal has more larger width than default one. (currently 820px)

      Use class modal-big to achieve this.

    • Responsive modal is streched both vertically and horizontally to the screen limits.

      Use class modal-responsive to achieve this.
